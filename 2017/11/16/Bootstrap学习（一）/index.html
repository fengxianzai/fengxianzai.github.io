Bootstrap官网

bootstrap全局css样式-排版

  • bootstrap将全局的字体大小设置为14px,行高设置为1.248，这些属性直接复制body，p标签被设置为等高为1/2行高，也就是10px的底部外边距。

  • 文本元素可以设置class为lead让其文本突出显示，文本中可以使用mark标签让其标记文字，设置特定样式。

  • 文本可以设置class为text-left、text-center、text-right让其文本居左、居中或者居右

  • bootstrap提供让其英文文本改变其大小写的功能，设置文本的class为text-lowercase，让文本全部小写，class为text-uppercase，让文本全部大写，class为text-capitalize,让文本首字母大写

  • bootstrap可以让文本中的文字设置缩略语，使用标签abbr，并设置title属性，其值为attribute

  • bootstrap可以使用address标签设置地址

  • bootstrap可以指定无样式列表，只需在ul标签中加上class，值为list-unstyled
